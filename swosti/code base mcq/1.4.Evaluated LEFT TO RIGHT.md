## **Rule: Operator Precedence (Left to Right)**

```java
// For addition and concatenation: LEFT TO RIGHT evaluation

// Example 1: Numbers first
10 + 20 + "30"
= (10 + 20) + "30"    // Group from left
= 30 + "30"           // Then concatenate
= "3030"

// Example 2: String first
"10" + 20 + 30
= ("10" + 20) + 30    // Group from left
= "1020" + 30         // Then concatenate
= "102030"
```
