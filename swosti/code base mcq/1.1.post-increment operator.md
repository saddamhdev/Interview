## **Understanding Post-Increment (`i++`)**

**`i++` has 2 parts:**
1. **Return current value** (before increment)
2. **Increment value** (after return)

```java
i = 5;
int x = i++;  // x gets 5 (current value)
              // i becomes 6 (after return)
```

## **Comparison: Pre-increment vs Post-increment**

### **With Post-increment (`i++`):**

```java
int i = 1;
while(i++ < 5);
System.out.println(i);  // 6
```

**Timeline:**
```
i = 1 â†’ compare 1 < 5 (TRUE) â†’ i = 2
i = 2 â†’ compare 2 < 5 (TRUE) â†’ i = 3
i = 3 â†’ compare 3 < 5 (TRUE) â†’ i = 4
i = 4 â†’ compare 4 < 5 (TRUE) â†’ i = 5
i = 5 â†’ compare 5 < 5 (FALSE) â†’ i = 6
Output: 6
```

---

### **With Pre-increment (`++i`):**

```java
int i = 1;
while(++i < 5);
System.out.println(i);  // 5
```

**Timeline:**
```
i = 1 â†’ increment to 2 â†’ compare 2 < 5 (TRUE)
i = 2 â†’ increment to 3 â†’ compare 3 < 5 (TRUE)
i = 3 â†’ increment to 4 â†’ compare 4 < 5 (TRUE)
i = 4 â†’ increment to 5 â†’ compare 5 < 5 (FALSE)
Output: 5
```

**Difference:** Pre-increment gives **5**, Post-increment gives **6** âŒ

---

## **Why the Difference?**

| Operator | Order | Result |
|----------|-------|--------|
| **`i++`** | Compare THEN increment | **6** |
| **`++i`** | Increment THEN compare | **5** |

## **Common Mistakes:**

### **Mistake 1: Forgetting the empty body**

```java
int i = 1;
while(i++ < 5);  // â† Empty body (semicolon)
System.out.println(i);  // 6

// vs

int i = 1;
while(i++ < 5)
    System.out.println(i);  // Prints 1,2,3,4
System.out.println(i);  // 5
```

---

### **Mistake 2: Confusing with for loop**

```java
// While loop: condition checked at START
int i = 1;
while(i++ < 5);  // Increments 5 times total

// For loop: similar
for(int i = 1; i++ < 5; ) { }  // Increments 5 times total
System.out.println(i);  // Error: i out of scope
```
