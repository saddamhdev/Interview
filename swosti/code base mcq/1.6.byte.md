দশ (10) সংখ্যাটি জাভাতে একটি বৈধ **byte** ভ্যালু।

জাভাতে `byte` ডাটা টাইপ সম্পর্কে কিছু গুরুত্বপূর্ণ তথ্য নিচে দেওয়া হলো:

### ১. রেঞ্জ (Range)

জাভাতে `byte` হলো একটি ৮-বিট সাইনড (signed) ইন্টিজার। এর সীমা বা রেঞ্জ হলো **-১২৮ থেকে ১২৭** পর্যন্ত। যেহেতু ১০ এই সীমার মধ্যে পড়ে, তাই এটি একটি সঠিক byte ভ্যালু।

### ২. মেমোরি এবং রিপ্রেজেন্টেশন

একটি `byte` মেমোরিতে মাত্র **১ বাইট (৮ বিট)** জায়গা নেয়। ১০ সংখ্যাটিকে বাইনারিতে প্রকাশ করলে তা হয় `1010` যা ৮ বিটের মধ্যে অনায়াসেই ধরে যায়।

---

### ব্যবহারিক উদাহরণ

আপনি যদি কোডে ১০-কে byte হিসেবে ডিক্লেয়ার করতে চান, তবে এভাবে লিখতে পারেন:

```java
byte b = 10; // এটি সঠিক

```

তবে মনে রাখবেন, জাভাতে যখন আপনি কোনো গাণিতিক অপারেশন করেন (যেমন `b + 1`), তখন জাভা স্বয়ংক্রিয়ভাবে সেই ভ্যালুটিকে `int`-এ রূপান্তর করে ফেলে। তাই তখন আপনাকে আবার কাস্টিং করতে হতে পারে:

```java
byte result = (byte) (b + 5); // এখানে (byte) দিয়ে কাস্ট করতে হয়

```

**আপনি কি জানতে চান কীভাবে ১২৭ এর বেশি কোনো সংখ্যাকে byte-এ রূপান্তর করলে কী ঘটে (ওভারফ্লো)?**
